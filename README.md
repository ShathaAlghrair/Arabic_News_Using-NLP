# Arabic_News_Using-NLP
ArabicSeek — End-to-end semantic search &amp; hybrid summarization for Arabic news. Dense retrieval (E5) + topic clustering (UMAP/HDBSCAN) + faithfulness-first summarization (TextRank+MMR). Achieves NDCG@5 0.79 (+18 pts over BM25), 0.19s CPU latency, 20 topics. Built with Python, Hugging Face, Streamlit.
